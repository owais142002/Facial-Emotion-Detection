# Facial Emotion Detection using YOLOv8

This project is a Flask web application that detects facial emotions in real-time using a YOLOv8 model. The model can detect the following emotions:
- Disgusted
- Surprised
- Angry
- Sad
- Happy
- Scared
- Neutral
